About EVADE

The access to EnVironmental Analytics for Developing countriEs (EVADE) project is hosted on the STFC/NERC JASMIN "super-data-cluster" facility (http://jasmin.ac.uk) and builds on STFC and NERC investments into the Community Intercomparison Suite (CIS - http://cistools.net) to develop a demonstration of an easy-to-use web portal providing access to environmental analytics of "big" Earth System data to users and scientists in developing countries (maximising co-benefits for the UK scientific and commercial applications).

This site is intended to demonstrate these concepts and explore the technical challenges in realising this ambition.

Demo

This site contains two main demonstrations: Synthesising and exploring data.

The Sythensis page shows how a user might create a new dataset by selecting a region and the satellite measurement to create a time series from. The region can either be a pre-defined mega-city, or a user-defined box. Clicking 'Submit' will send the user to the 'Jobs' page and show the compute job that would be created. The compute facility is unavailable for the demo for security reasons so the jobs presented here are examples only.

A set of pre-created measurement time-series can be explored in the Explore page. Here the user can click on any of the markers on the map to see a time series of the AOD in that location as measured the MODIS instrument. The user can also pan and zoom into the time series and save the resulting plot for offline use.

Funding

Funding for this initial demonstration phase was provided by STFC (ST/P003206/1). CIS is developed as open-source software and deployed for community use on the JASMIN platform provided by UK NERC, the UK Space Agency and the Science and Technology Facilities Council.

Design

As far as possible the platfrom uses a standard Django MVC design. We use the Djagngo-Rest-Framework (DRF) to create an API for the underlying data which the various views should use directly. We use Leaflet for the map components (linked to Mapbox for the basemap) and plot.ly for the time series plots.
